Vocabulaire

    isset() : Une fonction en PHP qui vérifie si une variable est définie et n'est pas nulle. Elle est souvent utilisée pour éviter les erreurs lors de l'accès à des variables qui pourraient ne pas être initialisées.

    trim() : Une fonction en PHP qui supprime les espaces en début et fin de chaîne de caractères. Elle est souvent utilisée pour nettoyer les données saisies par l'utilisateur.

    password_verify() : Une fonction en PHP qui vérifie si un mot de passe correspond à un hachage de mot de passe. Elle est utilisée pour authentifier les utilisateurs en comparant le mot de passe saisi avec le mot de passe stocké de manière sécurisée.

    header() : Une fonction en PHP qui envoie des en-têtes HTTP au client. Elle est souvent utilisée pour rediriger les utilisateurs vers une autre page ou pour spécifier le type de contenu à retourner.

    exit() : Une fonction en PHP qui termine l'exécution du script en cours. Elle est souvent utilisée après une redirection pour s'assurer que le reste du code ne s'exécute pas.

    session_start() : Une fonction en PHP qui démarre une session ou reprend une session existante. Elle est utilisée pour stocker des informations sur les utilisateurs à travers différentes pages d'un site web.

    htmlspecialchars() : Une fonction en PHP qui convertit les caractères spéciaux en entités HTML. Elle est utilisée pour prévenir les attaques de type Cross-Site Scripting (XSS) en échappant les caractères qui pourraient être interprétés comme du code HTML ou JavaScript.

    ENT_QUOTES : Une constante utilisée avec la fonction htmlspecialchars() en PHP. Elle indique que les guillemets simples et doubles doivent être convertis en entités HTML, ce qui renforce la sécurité contre les attaques XSS.

    PDO (PHP Data Objects) : Une extension de PHP qui fournit une interface pour accéder à différentes bases de données de manière sécurisée et efficace. Elle permet d'exécuter des requêtes SQL et de gérer les connexions à la base de données.

    prepare() : Une méthode de l'objet PDO en PHP qui prépare une requête SQL pour l'exécution. Elle est utilisée pour sécuriser les requêtes contre les injections SQL en utilisant des paramètres liés.

    bindParam() : Une méthode de l'objet PDO en PHP qui lie une variable à un paramètre de requête préparée. Elle est utilisée pour sécuriser les requêtes SQL en empêchant les injections SQL.

    execute() : Une méthode de l'objet PDO en PHP qui exécute une requête préparée. Elle est utilisée pour envoyer la requête SQL au serveur de base de données après que les paramètres ont été liés.

    fetch() : Une méthode de l'objet PDO en PHP qui récupère la prochaine ligne d'un ensemble de résultats de requête. Elle est utilisée pour accéder aux données retournées par une requête SQL.

    FETCH_ASSOC : Une constante utilisée avec la méthode fetch() de PDO en PHP. Elle indique que les résultats de la requête doivent être retournés sous forme de tableau associatif, où les clés sont les noms des colonnes de la base de données.
